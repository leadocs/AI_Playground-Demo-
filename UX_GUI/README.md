# AI Platform Experience Center - GUI Code

This directory contains the auto-generated frontend code for the "Visual Model" experience center, based on the Figma design.

## Structure

- `index.html`: The main HTML structure of the page.
- `styles.css`: The stylesheet defining the layout and visual styles.
- `assets/`: Contains the SVG images extracted directly from the Figma file.

## Features Implemented

- **Responsive Layout**: Sidebar + Main Content layout using Flexbox.
- **Visual Fidelity**: Colors, fonts, and spacing matched to the Figma design data.
- **Component Separation**:
  - Sidebar with navigation groups.
  - Top Header with actions.
  - Alert Banner for system notifications.
  - Central Workspace with "Welcome" state and Model Selector.
  - Input Area with shadow and tool icons.

## Usage

1. Open `index.html` in any modern web browser to view the page.
2. The page is static and represents the "Text Model" initial state as designed.
