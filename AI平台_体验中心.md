## 3. 文本模型
### 1. 二级分类（无）
- 厂牌
- 系列/版本
- 接入点（更新）

### 2. 主工作区
#### A-默认初始（生成前）
- **输入**
    - **[必]** 提示词
    - **[开关-默认关]** 深度思考
    - **[开关-默认关]** 联网
    - **[可选]** 图片
        - 支持张数（看模型决定）、UI区分（让用户意识到1/多）
        - 无图/有图状态、多图排列、悬停效果、支持点击删除
    - **[必]** API Key
        - 默认选中最近一次使用Key
        - 仅展示可用Key
- **输出**
    - 对话流

#### B-推理进行中
- **对话流**
    - 一期不调整，对齐Modelverse，仅做组件替换

### 3. 其他操作
- **[抽屉-默认关]** 模型参数
    - 一期不调整，对齐Modelverse，仅做组件替换
- 清空对话
- 模型对比
    - 点击后分2屏，共用提示词输入区域，独立模型对话流区域
    - 独立模型选择区域（默认同时选中对比前模型）
    - 开启后支持「退出对比」，即恢复默认但模型模式

## 4. 视觉模型
### 1. 二级分类
- 视觉理解 / 图片生成 / 视频生成

### 2. 模型选择
- 厂牌
- 系列/版本
- 接入点（更新）

### 3. 主工作区

#### 视觉理解
- **A-默认初始（生成前）**
    - **输入**
        - **[必]** 图片/视频
            - *这里重点设计：图片/视频二选一，上传一种格式后，能继续上传该种格式文件。清理已上传文件后可重新选择上传类型及新文件*
            - 支持多张（看模型）—待接口确认
            - 无图/有图状态、多图排列、悬停效果、支持点击删除
        - **[必]** 提示词
        - **[开关-默认关]** 深度思考
        - **[开关-默认关]** 联网
        - **[必]** API Key
            - 默认选中最近一次使用Key
            - 仅展示可用Key
    - **输出**
        - 对话流
- **B-推理进行中**
    - **对话流**
        - 「文本展示」一期不调整，对齐Modelverse，仅做组件替换
        - 「增加」图片展示，modelverse目前没有

#### 图片生成
- **A-默认初始（生成前）**
    - **输入**
        - **[选]** 图片
            - 支持多张（看模型）—待接口确认
            - 无图/有图状态、多图排列、悬停效果、支持点击删除
        - **[必]** 提示词
            - **[快捷]** 预设提示词
                - 一期不调整，对齐Modelverse，仅做组件替换
        - **[快捷-若有]** 比例-select
        - **[快捷-若有]** 图片数量-滑块/number-input
        - **[必]** API Key
            - 默认选中最近一次使用Key
            - 仅展示可用Key
    - **输出**
        - 1张图片
- **B-推理进行中**
    - 加载
- **C-推理完成**
    - 图片结果
    - 保留提示词输入、可以重新输入进行生成
    - **[操作]** 支持点击图片、视窗中心放大
        - 一期不调整，对齐Modelverse，仅做组件替换
    > *modelverse没有图库能力，一期不用参考方舟*

#### 视频生成
- **A-默认初始（生成前）**
    - **输入**
        - **[选]** 图片
            - 首帧、尾帧
                - 根据模型选择展示槽位
        - **[必]** 提示词
            - **[快捷]** 预设提示词
                - 一期不调整，对齐Modelverse，仅做组件替换
        - **[快捷-若有]** 时长-select
        - **[快捷-若有]** 分辨率-select
        - **[快捷-若有]** Prompt智能改写（?）-开关
            - ? 提示文案对齐Modelverse
        - **[必]** API Key
            - 默认选中最近一次使用Key
            - 仅展示可用Key
    - **输出**
        - 视频展示+任务序列
            - 「基本样式」一期不调整，对齐Modelverse，仅做组件替换
            - 「建议增加」单视频的关键属性标签（时长、分辨率、比例、有无音频、模型）
				>
- **B-推理进行中**
    - 展位卡加载（看方舟）
- **C-推理完成**
    - 除了任务序列单视频建议增加关键属性标签，其他与modelverse一致，仅做组件替换

### 4. 其他操作（与模型选择同行，居右）
- 清空对话
    - 仅图像理解有
- **[抽屉-默认关]** 高级参数
    - 「参数」一期不调整，对齐Modelverse，仅做组件替换
    - 「增加」重置
